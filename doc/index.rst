KNIME Python Script (Labs) API
==============================


This document lists the API of the module ``knime_io`` that functions as the main contact point between KNIME
and Python in the `KNIME Python Script (Labs) node <https://docs.knime.com/latest/python_installation_guide/index.html#_introduction_2>`_.
Please refer to the `KNIME Python Integration Guide <https://docs.knime.com/latest/python_installation_guide/index.html>`_ for more details on how to set up and use the node.
